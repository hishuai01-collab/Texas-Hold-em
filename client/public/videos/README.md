# 视频背景资源目录

此目录用于存放全局动态视频背景的资源文件。

## 文件说明

| 文件 | 说明 |
|------|------|
| `background.mp4` | 视频背景文件（H.264 编码，推荐 1080p 或 720p） |
| `poster.jpg` | 视频加载失败时的海报图片（回退显示） |

## 如何添加视频背景

### 方式 1：本地文件（推荐）

1. 将视频文件重命名为 `background.mp4`，放置在此目录下
2. 将海报图片命名为 `poster.jpg`，放置在此目录下
3. 视频将自动在应用启动时播放（静音循环）

### 方式 2：远程视频 URL

1. 在项目根目录的 `.env` 文件中添加：
   ```
   VITE_VIDEO_BACKGROUND_URL=https://your-video-url.com/video.mp4
   ```
2. 远程视频优先级高于本地文件

## 视频规范建议

- **格式**：MP4（H.264 视频 + AAC 音频）
- **分辨率**：1920×1080 或 1280×720
- **时长**：15-30 秒（无缝循环）
- **文件大小**：建议小于 5MB（影响加载速度）
- **内容**：建议使用美女真人背景，画面柔和，避免高频闪烁

## 美女真人背景视频推荐来源

- **Pexels 视频**（免费，无需署名）：https://www.pexels.com/videos/
- **Pixabay 视频**（免费，无需署名）：https://pixabay.com/videos/
- **Coverr**（免费，无需署名）：https://coverr.co/

## 注意事项

1. 移动设备默认不启用视频背景（省流耗电）
2. 视频必须静音才能自动播放（浏览器 autoplay 政策）
3. 如果视频加载失败，系统会自动回退到渐变背景
4. 系统会尊重 `prefers-reduced-motion` 设置，关闭动画
